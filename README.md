# 🚀 Raksha's Portfolio

This is my personal portfolio website built using **React + Vite + Tailwind CSS + Framer Motion**.  
It showcases my **skills, projects, and achievements** in an interactive way.

## 🌟 Features
- Beautiful animated hero section with typing effect
- Projects with GitHub + Live Demo links
- Responsive design (mobile-friendly)
- Smooth animations using Framer Motion
- Contact form with social links

## 🛠️ Tech Stack
- React
- Vite
- Tailwind CSS
- Framer Motion
- React Typed
